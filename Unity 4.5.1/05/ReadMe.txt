
■ 틱택토 샘플 게임 

● 실행 파일
05\TicTacToe\bin\TicTacToe.exe

● 플레이 방법
타이틀 화면에서 대전 상대를 기다릴 플레이어는 '대전 상대를 기다립니다'를 눌러주세요.
대기 중인 플레이어와 대전할 플레이어는 '상대방 IP 주소' 텍스트 필드에 대기 중인 플레이어의
IP 주소를 입력하고 '대전 상대와 접속합니다'를 눌러주세요.


'대전 상대를 기다립니다'를 선택한 플레이어가 선으로 '○'이고'대전 상대와 접속합니다'를 선택한  
플레이어가 '×'가 됩니다.

가로, 세로, 사선 중 하나를 채운 쪽이 승리입니다. 
10초 이내에 입력하지 않으면 랜덤하게 마크가 배치됩니다. 


● 예제 프로그램 
프로젝트 파일：05\TicTacToe\Assets\Scene\TicTacToe.unity
프로그램：05\TicTacToe\Assets\Script

Sequence.cs			게임 전체의 시퀀스 제어 
TicTacToe.cs			틱택토의 시퀀스 제어 및 통신 제어
TransportTCP.cs			TCP 소켓 통신 프로그램
